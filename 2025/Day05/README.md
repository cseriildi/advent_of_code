# Day 5

You can find the full task description and your input on [the Advent of Code website](https://adventofcode.com/2025/day/5).

## Short summary

A list of fresh ingredient ID ranges, and a list of available ingredient IDs are given.

### Example

```
3-5
10-14
16-20
12-18

1
5
8
11
17
32
```

### Part 1

Count how many of the available ingredient IDs are fresh.

### Part 2

Count all distinct fresh ingredient IDs. Availability doesn't matter.
