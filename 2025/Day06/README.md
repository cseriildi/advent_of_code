# Day 6

You can find the full task description and your input on [the Advent of Code website](https://adventofcode.com/2025/day/6).

## Short summary

We are given numbers and operators that should be applied on the numbers.

### Example

```
123 328  51 64
 45 64  387 23
  6 98  215 314
*   +   *   +
```

### Part 1

The numbers are separate by spaces and we have to apply the operators on each column and then sum the results together.

### Part 2

Now the digits of each number is to be read vertically, and the numbers for the calculation from right to left.
