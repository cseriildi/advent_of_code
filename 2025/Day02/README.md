# Day 2

You can find the full task description and your input on [the Advent of Code website](https://adventofcode.com/2025/day/2).

## Short summary

A list of product ID ranges is provided. We have to find invalid IDs.

### Example

```
11-22,95-115,998-1012,1188511880-1188511890,222220-222224,
1698522-1698528,446443-446449,38593856-38593862,565653-565659,
824824821-824824827,2121212118-2121212124
```

### Part 1

An ID is invalid if it can be formed by repeating a pattern exactly twice.
Sum all invalid IDs.

### Part 2

An ID is invalid if it can be formed by repeating a pattern any number of times.
Sum all invalid IDs.
