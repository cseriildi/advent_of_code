# Day 1

You can find the full task description and your input on [the Advent of Code website](https://adventofcode.com/2025/day/1).

## Short summary

We have a circular safe dial that rotates between 0-99. It starts at 50.
We are given a list of dial rotations with L/R direction and the number of clicks to turn.

### Part 1

Count how many times the dial ends a rotation pointing exactly at 0.

### Part 2

Count every time the dial passes or lands on 0 during rotations.
